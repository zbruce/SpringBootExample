# SpringBootExample
## 慕课网课程《两小时学会Spring Boot》

这是一个spring boot的web项目的demo。
- 使用多环境的不同配置，比如使用配置文件中的变量、属性配置
- controller的使用
- 使用jpa进行数据库操作，比如不写sql就可以自动建表、增删改查数据、数据库的事务操作

## 慕课网课程《Spring Boot进阶之Web进阶》

继上一课程后，对原有项目进行扩展。
- 表单验证，验证传入参数
- aop，拦截类和方法
- 统一异常处理，使用Result类封装返回结果，自定义异常类增加错误码，使用枚举管理错误码和错误信息
- 单元测试
- 记录日志
